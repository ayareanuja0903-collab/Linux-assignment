# 🐧 Linux Shell Scripting Assignment

This repository contains a collection of **Bash scripts** developed as part of a Linux assignment.
The scripts demonstrate automation, file handling, text processing, and system monitoring.

---

## 📂 Project Structure

```
.
├── count.sh                # Count words, lines, characters
├── search_string.sh       # Search string in file/directory
├── backup.sh              # Backup directory with timestamp
├── schedule_task.sh       # Schedule task using cron
├── replace_word.sh        # Replace word in file
├── realtime_monitor.sh    # Real-time log monitoring
├── data.txt               # Sample data file
└── README.md
```

---

## 🚀 Scripts Overview

### 1️⃣ Count File Statistics

Counts number of **lines, words, and characters** in a file.

```bash
./count.sh
```

---

### 2️⃣ Search String Recursively

Searches for a string in a file or directory.

```bash
./search_string.sh
```

---

### 3️⃣ Backup Script

Creates a compressed backup with timestamp.

```bash
./backup.sh
```

---

### 4️⃣ Cron Job Scheduler

Schedules tasks using cron.

```bash
./schedule_task.sh
```

---

### 5️⃣ Replace Word in File

Replaces all occurrences of a word in `data.txt`.

```bash
./replace_word.sh
```

---

### 6️⃣ Real-Time Monitoring Tool

Monitors logs in real-time and detects patterns.

```bash
./realtime_monitor.sh
```

## ⚙️ Requirements

* Linux OS (Ubuntu recommended)
* Bash shell
* Basic utilities: `grep`, `awk`, `sed`, `tar`, `cron`

## ▶️ How to Run

1. Clone the repository:

```bash
git clone <your-repo-link>
cd Linux-assignment
```

2. Give execute permission:

```bash
chmod +x *.sh
```

3. Run any script:

```bash
./script_name.sh
```
